# My Portfolio

A modern, responsive portfolio website showcasing my work as a Software Engineer and Innovator.

## Features

- 🎨 Modern dark/light theme with purple accent
- ⌨️ Animated typewriter effect
- 📱 Fully responsive design
- 🌐 SEO optimized
- 🎓 Education section with timeline
- 💼 Experience section with timeline
- 🎯 3 featured projects
- 💼 Services section
- 📄 CV/Resume download
- 📬 Contact form with social links

## Sections

- **Home** - Hero with profile image and typewriter effect
- **About** - Personal introduction and skills
- **Education** - Academic background (St. Joseph's School Rapogi 2020-2023, Zetech University 2025-2027)
- **Experience** - Work history
- **Services** - What I can do for you
- **Projects** - Featured projects (Leanlight, Teachable Machine, Be My Valentine)
- **CV** - Download resume
- **Contact** - Get in touch

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with CSS variables
- **JavaScript** - Interactive features
- **Font Awesome 6.5.1** - Icons
- **Google Fonts (Outfit)** - Typography

## Projects

1. **Leanlight** - A platform for students to access curated learning content and track their progress
2. **Teachable Machine** - An interactive image classification model built with Google's Teachable Machine
3. **Be My Valentine** - A fun and interactive Valentine's Day themed web application

## Getting Started

To run this project locally:

1. Clone the repository:
```bash
git clone https://github.com/Elly739/Portfolio.git
```

2. Open `index.html` in your browser

Or use a local server:
```bash
# Using Python
python -m http.server

# Using Node.js
npx serve
```
